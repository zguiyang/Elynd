
# 英语学习产品概述

## 1. 产品概念

这是一个 ​**AI驱动的英语阅读学习应用**​，核心围绕 ​**公共领域（Public Domain）经典英文书籍**。

例如：

- 《Pride and Prejudice》
- 《Alice in Wonderland》
- 《The Adventures of Sherlock Holmes》

这些书籍来自：

- Project Gutenberg

产品核心理念：

> **通过一本完整书籍，实现沉浸式英语学习。**

不同于传统英语学习产品（碎片化课程或单词背诵），该产品强调：

**深度阅读 + AI辅助理解 + 循环复习。**

---

# 2. 目标用户

主要用户群：

**成人英语学习者**

年龄：

25–45岁

需求：

- 职场英语
- 移民英语
- 长期自学

用户特点：

- 不喜欢碎片课程
- 更喜欢阅读真实内容
- 希望提升词汇和阅读能力

---

# 3. 核心学习模型

产品的学习逻辑基于一个简单循环：

```
阅读

↓

```

|阶段|功能|
| ------| ----------|

|理解|AI解释|
|练习|练习题|

# 5. 单词学习系统

这是整个产品​**最核心的学习模块**。

核心思想：

不是词典，而是：

**书籍词汇系统**

---

## 5.1 书籍词汇分析（离线）

在后台预处理整本书。

处理流程：

```
书籍文本
↓
分词
↓
词形归一
↓
词频统计
↓
CEFR等级
```

---

### 1 提取单词

tokenization

示例：

```
Alice was beginning to get very tired
```

变成：

```
["alice","was","beginning","to","get","very","tired"]
```

---

### 2 词形归一

例如：

```
run
runs
running
ran
```

归一为：

```
run
```

工具：

- WordNet

Node库：

- wink-lemmatizer
- natural

---

### 3 词频统计

统计每个词出现次数。

结果示例：

```
alice: 403
rabbit: 120
queen: 85
```

---

### 4 CEFR等级

CEFR是欧洲语言等级体系：

|等级|词汇规模|
| ------| -----------|
|A1|\~1000|
|A2|\~2000|
|B1|\~4000|
|B2|\~6000|
|C1|\~8000|

词库来源：

- Cambridge English

数据格式：

```
word,cefr
run,A1
beautiful,A2
consequence,B2
```

程序匹配：

```
cefr = cefrDict[word]
```
